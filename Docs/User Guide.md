**User Guide**

This guide is for the potential players of BoardSmith. BoardSmith is a custom board game that allows you to create your own game by combining various rules and creating your own board. You can then play your custom games with your friends online. To run BoardSmith, simply download our built version and run the .exe. Once you’re in the main menu you have two paths you could take. You can join a game a friend is hosting or you can create your own custom game. 

**Installation:**  
To install BoardSmith, you’ll simply download a built version of the game, and then run the .exe that's in the BoardSmith folder.

**Game Creation:**  
If you choose to create your own game, you’ll be directed to our editor. Here you’ll start at the rule editor, but you can freely switch between the board and rule editor at any time. In the rule editor you’ll look through various rules, such as currency, combat, number of dice, etc.. You’ll be able to combine these rules as you see fit to create a unique ruleset. In the board editor you’ll be able to create and customize your tiles, changing things like: tile size, shape, colors/textures and position. You can also change the flow of movement along the tiles. Boards can be linear with players moving only in one direction, or you can make more complex boards that allow for movement in reverse or even tiles that point to 2+ other tiles. Once you’ve finished creating a custom ruleset and board, you’ll save it and then you’ll be able to host a game with your settings. 

**Hosting:**  
To host a game, you’ll either need to have a saved custom game, or you can host one of our premade games. Simply pick a game from your list of saved games and a lobby will be created with you as the host. Share the invite code with your friends and they will be able to join. Once your friends are in the lobby with you, you’re ready to start. 

**Joining:**  
To join a game, you’ll need to get an invite code from a friend who’s hosting a game. Simply enter this code in the join lobby field and you’ll be automatically connected to their lobby. Wait for the host to start the game.

**Gameplay:**  
To avoid your friends having to download the custom content, the game will run through the host entirely. This does come with a drawback; if the host leaves, the game is over. Once the game starts, take turns rolling dice and whatever else pertains to the custom game. Play through the game until end conditions are met. 

**Outputs:**  
Outputs will be displayed as messages in the UI, and should clearly relay information. 

**Troubleshooting:**  
If you are trying to share a game you’ve made with a friend but they don’t see it in their list of saved games, make sure you are placing the custom game file in the right location. A restart of BoardSmith might be needed after files have been added for the list to correctly load. Keep in mind, your friend does not need to share the game for you to play together, this is only if you want to be able to host that game yourself.
